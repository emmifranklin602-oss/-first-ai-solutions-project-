# AI Safety & Accuracy System

This project demonstrates a combined prototype for:

## 🛡️ AI Safety Features
- Live selfie capture for identity verification
- Device fingerprinting
- Behavioral analysis

These features simulate systems that prevent fake accounts and harmful activity.

## 🤖 AI Accuracy Tracking
- Users can input sample values
- The system predicts closest values from dataset
- It tracks predictive accuracy over time

## 📌 How to Use
1. Open the web app
2. Allow camera access
3. Test identity verification
4. Enter sample values to test AI prediction accuracy
